Detailed Description

The AAudio API was not available in early versions of Android. To avoid linker errors, we dynamically link with the functions by name using dlsym(). On older versions this linkage will safely fail.

◆ close()

int oboe::AAudioLoader::close ( )

Close the AAudio shared library. This can be called multiple times. It should only be called from one thread.

The open() and close() do not nest. Calling close() once will always close the library. The destructor will call close() so you don't need to.

Returns
0 if successful or negative error.
